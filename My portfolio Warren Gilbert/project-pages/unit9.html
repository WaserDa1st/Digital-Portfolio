About this course
In this unit we got a strong understanding on how spreadsheets are used to store, manipulate and analyse to present it in easy-to-understand ways we then later discovered the many tools and techniques that are available in most spreadsheet software and learn how to correctly use them and apply them in spreadsheets lastly, we then went on investigate some of the ways spreadsheets are used in the real world.
Topics Learnt / Skills that I have Developed

-VLOOKUP formula

-Conditional formatting

-Data validation

-Cell formatting

-COUNTIF formula

-Sorting data charts

The skills that I have developed within UNIT 9 spreadsheet development are how to correctly use the VLOOKUP formula in my spreadsheets to help improve efficiency and productivity in the spreadsheets. The next skill is developed was cell formatting and how this is used for efficient data management, improved data visualization and easy data referencing. The last skill I developed is when and where to use data validation within my spreadsheets and how using data validation restricts data entry to certain cells.
